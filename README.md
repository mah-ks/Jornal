# Jornal
Descrição do Projeto – Jornal 

Este projeto consiste em um jornal digital que apresenta notícias de tecnologia do cotidiano, destacando tendências e inovações. Ele possui a seção Apresentações, onde é exibido o Woman in Action, mostrando os trabalhos em grupo das alunas, que abordaram temas voltados para o desenvolvimento pessoal e profissional, com o objetivo de ajudarmos umas às outras.

Além disso, o projeto inclui links para as principais notícias do The News e conta com uma seção de notícias contendo meu comentário e opinião sobre cada tema abordado.
